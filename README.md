## Первая практика по предмету "Фронтэнд и бэкэнд разработка"
### Практическое задание
Необходимо реализовать карточку товара (содержимое: название, описание,
фотография) с применением переменных (минимум 2), миксина (минимум 1) и
использованием вложенной структуры селекторов на любом из двух препроцессоров
(SASS или LESS).
